# 西洋跳棋

一个浏览器版西洋跳棋游戏。当前版本是完整独立项目，不包含外脑模块，也不会请求任何大模型 API。

## 已实现

- 8x8 西洋跳棋棋盘
- 暗格行棋、必须吃子、连吃、升王
- 人机模式，内置 AI 分为轻松、稳健、强手
- 同屏双人模式
- 房间联机模式
- 刷新后自动尝试回到原房间

## 启动

双击 `启动游戏.command` 可以本机试玩。

也可以在终端运行：

```bash
cd /Applications/My_VSCode_project/2026/6/25/西洋跳棋
node server.mjs
```

打开：

```text
http://localhost:5186
```

同一局域网联机时，双击 `局域网联机.command`，再让其他设备打开你的电脑局域网地址加端口。

## 部署

这个项目可以部署到支持 Node.js 的 Web Service 平台。服务会读取平台提供的 `PORT` 环境变量。

Render 可直接使用仓库里的 `render.yaml`，启动命令是：

```bash
node server.mjs
```

## 规则

- 墨方先手，从底部向上走；朱方从顶部向下走。
- 普通子只能向前斜走，王可以前后斜走。
- 有吃子时必须吃子。
- 吃完后如果同一枚棋子还能继续吃，必须连吃。
- 普通子到达对边后升王；升王时本回合结束。
- 吃光对方棋子，或让对方无合法走法，即获胜。
